# 農業勤怠管理システム 仕様書（NFC版）

NFCタグを従業員証として利用し、管理者スマホで簡単に勤怠を記録するシステムです。

## 1. システム構成

- **フロントエンド**: GitHub Pages (HTML/JavaScript)
  - Web NFC API を使用して管理者のスマートフォンでタグを読み取ります。
- **バックエンド**: Google Apps Script (REST API)
  - データの保存、ステータス管理、給与計算を行います。
- **データベース**: Google スプレッドシート
  - 勤怠データおよびスタッフ情報を管理します。

## 2. 主要機能

### 2.1 打刻機能（リーダーモード）
- **自動判定**: 従業員がNFCタグをスマホにかざすと、現在のステータス（未出勤/勤務中/休憩中）に応じて「出勤・退勤・休憩終了」を自動で判定し記録します。
- **フィードバック**: 打刻成功時に名前・区分・時刻を画面表示し、バイブレーションで通知します。
- **重複防止**: 同じタグの連続読み取りを3秒間ブロックするクールダウン機能を搭載。
- **時刻端数処理**: 打刻時刻は5分単位で切り捨て処理されます（例: 08:04 → 08:00）。

### 2.2 スタッフ登録（登録モード）
- **かんたん登録**: 未登録のNFCタグをかざし、名前と時給を入力するだけで即座に運用を開始できます。
- **重複チェック**: すでに他のスタッフに紐付いているタグの重複登録を防止します。

### 2.3 管理者機能
- **稼働状況確認**: 全スタッフの現在のステータス（勤務中/休憩中/未出勤）をリアルタイムで一覧表示します。
- **履歴参照**: 本日の打刻履歴をリスト形式で表示します。
- **データ出力**: 勤務記録をCSV形式でダウンロードできます。
- **接続設定**: 管理者メニューからGASのAPIエンドポイントURLを設定・更新できます。

### 2.4 自動計算ロジック
- **実働時間**: （退勤時刻 - 出勤時刻）- 休憩時間 で算出。
- **給与計算**: 実働時間 × 時給 で算出。
- **自動休憩**: 12:00〜13:00を跨ぐ勤務の場合、手動休憩の記録がなければ設定された時間（デフォルト60分）を自動控除します。

## 3. 技術仕様

- **動作環境**: Android スマートフォン（NFC搭載端末）
- **推奨ブラウザ**: Google Chrome（Android版 89以降）
  - ※iOS（iPhone）およびPCブラウザはWeb NFC API非対応のため利用不可。
- **セキュリティ**: 
  - 通信はHTTPSで暗号化。
  - 個人データはすべてGoogleスプレッドシート（非公開）に保存。

## 4. データ構造（スプレッドシート）

### スタッフ名簿
- NFCシリアル, ID, 名前, 時給, 現在のステータス, 管理者フラグ

### 勤務記録
- 日付, NFCシリアル, ID, 名前, 区分, 時間, 実働時間, 給与, 備考

### 設定マスタ
- 自動休憩時間の設定、LINE通知用アクセストークン
